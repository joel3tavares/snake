# Snake Game

This project demonstrates the implementation of a doubly linked list through a simple snake game.

## Requirements

- CMake
- make
- gcc

## Build

Follow these steps to build and run the project:

1. **Compile dependencies**

    ```sh
    make deps
    ```

2. **Compile the project**

    ```sh
    make
    ```

3. **Run the game**

   ```sh
   $ make run
   ```
